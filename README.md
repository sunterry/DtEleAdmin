# DtEleAdmin
vue, vuex, vue-router, element-ui搭建的通用后台管理系统

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```


### Compiles and minifies for test
```
npm run stage
```

### Lints and fixes files
```
npm run lint
```
